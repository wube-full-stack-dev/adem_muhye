# 🥤 AdemCoca Beverage Distribution System

A full-stack beverage distribution management system with role-based access control, 3D animations, and real-time sales tracking.

## 🚀 Features

- 🔐 **Authentication** - JWT-based login/register with role-based access
- 👑 **Role Management** - Admin, Manager, and Staff roles with different permissions
- 📊 **Sales Management** - Add, view, and delete sales records
- 🛒 **Product Catalog** - Browse products by category (Soft Drinks, Juices, Water)
- 🎬 **3D Animated Background** - Immersive Three.js background with floating bottles
- 📱 **Responsive Design** - Works on desktop, tablet, and mobile
- 🗺️ **Location Map** - Interactive Google Maps integration
- 💬 **Contact System** - WhatsApp, Telegram, and phone support

## 🛠️ Tech Stack

### Frontend
- React 19
- Tailwind CSS
- Three.js (3D animations)
- React Router DOM
- Bootstrap 5
- Vite

### Backend
- Node.js
- Express
- MySQL
- JWT Authentication
- Bcrypt Password Hashing

## 📋 Prerequisites

- Node.js (v18+)
- MySQL (v8+)
- npm or yarn
