# 🥤 Sales Management System

A full-stack sales management application built with React, Node.js, Express, and MySQL.

## ✨ Features

- ✅ User Authentication (Register/Login)
- ✅ Password hashing with bcrypt
- ✅ Add new sales with category-product filtering
- ✅ View all sales in responsive table
- ✅ Delete sales with confirmation
- ✅ Protected routes
- ✅ RESTful API architecture
- ✅ Bootstrap 5 UI
- ✅ Environment configuration

## 🏗️ Tech Stack

### Frontend
- React 18
- React Router DOM 6
- Bootstrap 5
- Vite
- Environment Variables

### Backend
- Node.js
- Express
- MySQL2
- bcrypt
- CORS
- dotenv

### Database
- MySQL
